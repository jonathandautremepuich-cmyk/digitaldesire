# Site de vente de formation

Landing page pour vendre une formation en ligne. Design sobre (fond sombre, accents dorés), sections : programme, avantages, tarifs, contact.

## Lancer en localhost

**Option 1 — avec Node.js :**
```bash
npx serve .
```
Puis ouvrir : http://localhost:3000

**Option 2 — avec Python 3 :**
```bash
python -m http.server 8080
```
Puis ouvrir : http://localhost:8080

**Option 3 — sans serveur :**  
Ouvrir directement `index.html` dans le navigateur (certaines fonctionnalités peuvent être limitées).
